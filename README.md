# Svelte + Vite + Windi

This template should help get you started developing with Svelte in Vite, styled by WindiCSS.

## Setup

```bash
npm degit reepolee/svelte-vite-windi#main svelte-app
cd svelte-app
npm i
npm run dev -- --open
```
